# Pir-Sensor-Node.js
PIR Sensor for Raspberry Pi (motion detecting)

# onoff

GPIO access and interrupt detection with **Node.js** on Linux boards like the
Raspberry Pi,

onoff supports Node.js versions 0.10, 0.12, 4, 5, 6, 7, 8 and 9.
